## Spacexl-api
### Author: Nenad Kostic

Spin up the backend locally with docker-compose
```sh
docker-compose up -d
```
This command will stop all containers
```sh
docker-compose stop
```
This command will stop and destroy all containers
```sh
docker-compose down
```

Seeding
```sh
npm run seed
```

Testing (Stop the api before running this command)
```sh
npm run test
```
